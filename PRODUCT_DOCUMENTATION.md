# Relate App 产品文档

## 1. 产品概述

Relate App 是一款专为现代人设计的人际关系管理应用，旨在帮助用户更好地维护和发展个人与职业关系。通过直观的界面和实用的功能，用户可以轻松跟踪互动、管理重要日期，并保持与联系人的良好沟通。

### 1.1 产品目标

- 帮助用户建立和维护健康的人际关系
- 提供便捷的工具记录和分析互动历史
- 提醒用户重要的日期和纪念日
- 简化沟通流程，提高沟通效率
- 可视化展示关系健康状况

### 1.2 目标用户

- 注重人际关系维护的个人用户
- 需要管理大量联系人的商务人士
- 希望改善社交技能的用户
- 希望保持与亲友联系的用户

## 2. 核心功能

### 2.1 签到功能 (Check-ins)

- **功能描述**：记录与联系人的每次互动，包括时间、地点、互动内容和感受
- **使用场景**：会面后立即记录，或回顾时补记
- **核心价值**：建立互动历史，分析关系发展趋势
- **操作流程**：选择联系人 → 填写互动详情 → 保存记录

### 2.2 日期管理 (Dates)

- **功能描述**：管理重要的日期和纪念日，如生日、周年纪念等
- **使用场景**：添加重要日期，接收提醒通知
- **核心价值**：避免忘记重要日期，提前准备庆祝或问候
- **操作流程**：添加日期 → 设置提醒 → 查看日期列表

### 2.3 消息模板 (Templates)

- **功能描述**：创建和使用标准化的消息模板，适用于不同场景
- **使用场景**：发送生日祝福、节日问候、感谢信息等
- **核心价值**：提高沟通效率，确保消息质量
- **操作流程**：创建模板 → 选择模板 → 编辑发送

### 2.4 关系温度计 (Thermometer)

- **功能描述**：通过可视化的温度计展示关系健康状况
- **使用场景**：快速了解关系状态，识别需要关注的关系
- **核心价值**：直观展示关系健康度，促进用户主动维护关系
- **操作流程**：查看主屏幕的温度计 → 根据提示采取行动

### 2.5 主题定制 (Theming)

- **功能描述**：自定义应用的外观，选择不同的主题
- **使用场景**：根据个人喜好或心情更换主题
- **核心价值**：提升用户体验，增加应用的个性化程度
- **操作流程**：进入设置 → 选择主题 → 应用更改

## 3. 用户界面设计

### 3.1 设计风格

- **整体风格**：现代、简洁、友好
- **色彩方案**：主色调为蓝色系，搭配中性色
- **排版**：清晰的层次结构，易读的字体
- **图标**：简约现代的线性图标

### 3.2 页面结构

- **主屏幕**：显示关系温度计和快捷操作
- **标签页导航**：Checkins、Dates、Templates 三个主要功能
- **详情页**：显示单个记录的详细信息
- **设置页**：应用配置和主题选择

### 3.3 交互设计

- **手势操作**：支持滑动、点击、长按等常见手势
- **动画效果**：平滑的过渡动画，增强用户体验
- **反馈机制**：操作后有明确的视觉和听觉反馈

## 4. 技术架构

### 4.1 技术栈

- **开发语言**：Swift 5
- **UI 框架**：SwiftUI
- **数据存储**：Core Data
- **响应式编程**：Combine
- **部署平台**：iOS 14.0+

### 4.2 系统架构

- **MVVM 架构**：Model-View-ViewModel
- **模块化设计**：功能模块清晰分离
- **数据流程**：单向数据流，易于追踪和调试

## 5. 业务流程

### 5.1 用户注册与登录

- **注册流程**：首次启动 → 欢迎页面 → 创建账户 → 完成设置
- **登录流程**：启动应用 → 输入凭据 → 进入主界面

### 5.2 核心业务流程

1. **记录签到**：
   - 选择联系人 → 填写互动详情 → 保存 → 查看历史记录

2. **管理日期**：
   - 添加日期 → 设置提醒 → 接收通知 → 查看日期列表

3. **使用模板**：
   - 创建模板 → 选择联系人 → 选择模板 → 编辑发送

4. **查看关系状态**：
   - 打开应用 → 查看温度计 → 分析状态 → 采取行动

## 6. 市场分析

### 6.1 竞争分析

- **现有解决方案**：通讯录应用、日历应用、笔记应用
- **Relate App 优势**：专注于人际关系管理，整合多种功能，提供独特的关系温度计

### 6.2 市场机会

- **市场需求**：现代人面临人际关系维护的挑战，需要工具帮助管理
- **目标市场**：25-45岁的职业人士和社交活跃人群
- **增长潜力**：随着人们对心理健康和人际关系的重视，市场需求将持续增长
